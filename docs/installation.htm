Important Notes:

LOX mod requires that Quake II (at least version 3.20) be installed for Client installation. We also encourage and recommend using the R1Q2 enhanced Quake 2 client.


Client installation:

  • Step #1

    • Download LOX installation files below:

    1.12.7 Full installation (WARNING - overwrites LOX folder cfg files- recommended for new installations)

    1.12.7 Update (For those who only need to update the game DLL; package contains both windows and linux files.)

  • Step #2

    • Unzip lox_1_12_7_full.zip into your C:\Quake2 directory (or where ever you have your base Quake 2 installation). The file unzips it's contents into a directory named lox.

    Now, to start, launch Quake and type "set game lox" in the console. You can now start a single-player game, or a deathmatch server, or whatever you want to do.
