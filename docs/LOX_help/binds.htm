use command: "invis" to become invisible

use command: "cloak" to cloak from scanner

use command: "tracker" a head 'gib' that shoots weapon

use command: "airstrike" to call down a hail of 10 missiles from the sky

use command: "holoquad" to make a fake quad damage

use command: "rangefinder" shows distance from you to target

use command: "checkstats" shows others health and armor stats

use command: "sweeperfiremode" to toggle the sweeper fire mode

use command: "explosive" to toggle the grenades explosive mode

use command: "detonate" to detonate the explosive

use command: "logplayers" logs players ip's and current pings

use command: "ver" find out what version of lox you are using

use command: "guided nuke" a guided Railgun nuke

use command: "guided missile" to use a Guided Missile

Bind a key to the command: "use plasma cluster grenade; use guided missile" to use a Grenade tipped Guided Missile.
