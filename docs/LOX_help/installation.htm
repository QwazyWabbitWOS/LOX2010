Important Notes:

LOX requires Quake II 3.20 for Client installation. 


Client installation:

Download the LOX installation file available at

1.12.7 Full installation (WARNING - overwrites LOX folder cfg files- recommended for new installations)

          1.12.7 Update (For those who only need to update the DLL)

 

Server installation:

For the Linux server, 3.21 is essential for proper operation and it's advisable for windows to upgrade to  3.21

LOX is linked with GLIBC 2.1.3 on Linux so any Linux server should use a build of the 3.21 dedicated server code that is linked with those libraries.
